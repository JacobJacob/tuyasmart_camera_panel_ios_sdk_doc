前言

涂鸦智能摄像机面板SDK iOS版接入指南

涂鸦智能摄像机面板SDK（简称：TuyaSmartCameraPanelSDK）为方便用户减少摄像头面板开发成本，对外提供了摄像头功能操作相关的多个面板。

TuyaSmartCameraPanelSDK 是在 TuyaSmartCameraSDK 基础上开发的一系列面板，支持用户选择是否接入，若不需要，也可自行开发（ 参考 TuyaCameraSDK 开发文档 ）
